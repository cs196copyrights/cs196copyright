Survey for UP Diliman Students

We've been assigned to do a survey about the opinions of UP Diliman Students, particularly those pursuing Engineering and Science degrees, regarding software copyrights and patents and the extent of influence a particular piece of design, code or technology has on a whole product.

This survey is undertaken as part of CS 196, a course on ethical and professional issues in computing at the Department of Computer Science, during the first semester of Academic Year 2016-2017 in UP Diliman.

Topic Description

Copyrights and Patents. We examine two cases, Apple vs Samsung and Google vs Oracle. In the Apple vs Samsung case, should one design infringement be equivalent to the the worth of the entire product? In Google vs Oracle, are source codes unique?

Files in the Repository

The repository contains a Word XML file, which is a print-friendly version of the survey form. Survey results, analyses and presentations (in TeX format) are also available in the repository. There is also a (now closed) online version of the form.

TeX files

The report is written in LaTeX and compiled using pdflatex from the MikTeX Windows implementation. It also uses a modified class file and some style files from the Association for Computing Machinery. The changes are only minor and are documented in the class file.

Survey Responses

The raw survey results are available in the Excel file results_charted.xlsx. This workbook contains several worksheets, mostly containing Pivot Tables used for data analysis. The raw data is found in the worksheet labeled "Raw Data". The number of male and female respondents are equal. Note that to achieve this, some respondents were arbitrarily dropped in order to equalize the numbers. The worksheet contains all responses (from both online and print forms).
